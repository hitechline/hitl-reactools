# usePosition

Have the [position](https://developer.mozilla.org/en-US/docs/Web/API/DOMRect) always the specified element to be updated.

<br />
<br />

## Usage

```tsx
import { usePosition } from '@hitechline/reactools';

const Component = () => {
  const { ref, ...position } = usePosition();

  return (
    <div ref={ref}>
      My position is <span>{JSON.stringify(position)}</span>
    </div>
  );
};
```
