# Installation

<br />

### Yarn

```shell
$ yarn add @hitechline/react@latest
```

<br />

OR

<br />

### NPM

```shell
$ npm install @hitechline/react@latest
```

<br />
<br />

##### How to use ?

```tsx
import * as Reactools from '@hitechline/react';

const Component = () => {
  return (
    <h1 className={Reactools.mergeClassNames('class1', 'class2')}>Component</h1>
  );
};
```
